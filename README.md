Inspiration & Technical Notes
-----------------------------

This project is inspired by the classic 1980s arcade game Frogger. The core mechanic — guiding a frog across a busy road and a treacherous river to safety — is translated here with lanes of traffic, floating logs, and five goal lily pads. The game is implemented in JavaScript using the q5/q5play-friendly template, includes a Title screen, Active Gameplay, and Game Over screen, and stores the high score and last run score in localStorage. Visuals lean into an 80s arcade aesthetic (retro starfield, grid perspective, and neon text), and the player character is represented with a frog emoji for a playful, minimalist look.
